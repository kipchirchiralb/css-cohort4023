CSS Colors

Relative units

These are ways of representing sizes/pixels in relation to existing values

They include: %(percentage), em, rem, vw and vh

  • %(percent) - This size is relative to the parent element e.g. If you set an element's width to 50%, it will occupy half of its parent container's width.
  • em - This is relative to the font size of the element. For example, if an element has a font size of 16px, then 1em is equal to 16px for that element. 2em would be 32px, and so on.
  • rem (root em) - Relative to the font size of the root element (html). If the root element's font size is 20px, then 1rem equals 20px across the entire document.
  • vw stands for "viewport width", and vh stands for "viewport height".10vw is equal to 10% of the viewport's width, and 50vh is equal to 50% of the viewport's height.
