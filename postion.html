CSS Position property

With the position property, we have five options

  • Static (default value)
  • Fixed -- allows positioning of an element in relation to the viewport
  • Relative -- allows positioning of an element in relation to its static position
  • Absolute-- allows positioning of an element in relation the first parent element with position set to relative , if none it behaves like position fixed
  • Sticky
