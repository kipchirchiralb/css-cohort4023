What is Syntax

Syntax refers to the set of rules that define the combinations of symbols considered to be correctly structured in a particular programming or scripting language, or more broadly, any formal language. It dictates how statements and expressions must be written so that they can be correctly executed or processed.

What is CSS

CSS stands for Cascading Style Sheets and is used to style and layout web content(html elements). The syntax of CSS is relatively straightforward and consists of selectors, properties, and values.
Here's a basic breakdown of the CSS syntax:

  1. Selector

    This determines which HTML elements the style will be applied to.

  2. Property

    This indicates the type of style attribute you wish to change (e.g., color, font-size, width).

  3. Value

    This is the actual style value you wish to set for that property.

Example

                
                    header {
                        text-align: center;
                        padding: 50px 0;
                    }

Where we write our CSS

  • Inline CSS

    Writing CSS directly within the HTML element using the style attribute.

  • Internal (or Embedded) CSS

    Writing css Within the head section of the HTML file using the style tag.

  • External CSS

    By linking to an external .css file. This is the most common method for larger websites as it promotes separation of content (HTML) and design/styling (CSS).
